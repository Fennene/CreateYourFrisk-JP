NewAudio オブジェクト [E/M/W] NewAudioオブジェクトを使用することで、オーディオチャンネルを作成し、Audioオブジェクトよりも優れた音楽の制御が可能になります。

オーディオチャンネルは、一度に最大１つの音楽/サウンドが再生できます。

注意! Audioオブジェクトで使用されているオーディオチャンネルは"src"と名付けられています。
戦闘の開始時に存在し、デフォルトではすべての音楽とサウンドが再生されます。
以下の関数にある"name"に指定できます。

  • NewAudio.CreateChannel( name) - nameという名前のオーディオチャンネルを作成します。

  • NewAudio.DestroyChannel( name) - nameという名前のオーディオチャンネルを破棄します。

  • NewAudio.Exists( name) - nameと言う名前のオーディオチャンネルが存在するか調べます。

  • NewAudio.GetAudioName( name, withPrefix = true) - Gets the name and type of the audio clip currently playing in the channel name as a string.
    If no audio is playing in a channel, this function returns "empty".

    If withPrefix is true, the string returned by this function will begin with music: for music clips, sound: for sound clips and voice: for voice clips.

    For example, playing the sound dogsecret in an audio channel and using this function on that channel would return the string "sound:dogsecret" (or "dogsecret").

  • NewAudio.GetTotalTime( name) - nameと名付けられたオーディオチャンネルの、最後に再生したオーディオ合計再生時間を返します。

  • NewAudio.GetPlayTime( name) or NewAudio.GetCurrentTime( name) - このチャンネルの、現在再生されているオーディオの現在の再生時間を返します。

  • NewAudio.PlayMusic( name, music, loop = false, volume = 1) - nameと名付けられたオーディオチャンネルでmusicを再生します。

    loopは音楽をループさせるかどうか表します。
    volumeは音量を表します。

    音楽はAudioフォルダにある.oggまたは.wavファイルを再生します。拡張子は含めないでください。

  • NewAudio.PlaySound( name, sound, loop = false, volume = 0.65) - nameと名付けられたオーディオチャンネルでsoundを再生します。

    loopはサウンドをループさせるかどうか表します。
    volumeは音量を表します。

    サウンドはSoundsフォルダにある.oggまたは.wavファイルを再生します。拡張子は含めないでください。

  • NewAudio.PlayVoice( name, voice, loop = false, volume = 0.65) - Plays the voice voice on the channel name.

    loop tells the engine if it should loop the voice or not;
    volume is the volume of the voice.

    Voices played are loaded from your mod's Sounds/Voices folder, as .ogg or .wav files. Don't include the file extension.

  • NewAudio.SetPitch( name, value) - nameと名付けられたオーディオチャンネルの、ピッチ(と速度)を設定します。
    1.0 で通常のピッチ/速度です。

  • NewAudio.GetPitch( name) - nameと名付けられたオーディオチャンネルの、ピッチを返します。

  • NewAudio.SetVolume( name, value) - nameと名付けられたオーディオチャンネルの、音量を設定します。
    0.75 がデフォルトの音量です。

  • NewAudio.GetVolume( name) - nameと名付けられたオーディオチャンネルの、音量を返します。

  • NewAudio.Play( name) - nameと名付けられたオーディオチャンネルで、最後に再生した音楽/サウンドを再生します。

  • NewAudio.Stop( name) - nameと名付けられたオーディオチャンネルで、音楽/サウンドの再生を停止します。

  • NewAudio.Pause( name) - nameと名付けられたオーディオチャンネルで、音楽/サウンドの再生を一時停止します。

  • NewAudio.Unpause( name) - nameと名付けられたオーディオチャンネルで、音楽/サウンドの再生の一時停止を再開します。

  • NewAudio.SetPlayTime( name, value) - nameと名付けられたオーディオチャンネルの、トラック位置(再生位置)を秒単位で指定します。

  • NewAudio.isStopped( name) - nameと名付けられたオーディオチャンネルの、音楽/サウンドの再生が停止(または一時停止)しているか調べます。

  • NewAudio.StopAll() - 全てのオーディオチャンネルの音楽/サウンドの再生を停止します。

  • NewAudio.PauseAll() - 全てのオーディオチャンネルの音楽/サウンドの再生を一時停止します。

  • NewAudio.UnpauseAll() - 全てのオーディオチャンネルの音楽/サウンドの再生の一時停止を再開します。
