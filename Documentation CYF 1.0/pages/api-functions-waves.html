Arena オブジェクト



次のセクションはアリーナ専用です。[???](The following section is dedicated exclusively to the arena.[原文])
アリーナとは、プレイヤーが移動できる白いボックス内の領域を表します。
発射物(Projectile)はアリーナに対してスポーンして移動でき、アリーナのサイズを変更して移動できます。[翻訳力不足]
(Projectiles can spawn and move relative to the arena, and the arena can be resized and moved.[原文])


Arena オブジェクト [W] このオブジェクトを使用して、アリーナに関する情報を取得したり、大きさを変えたり移動できたりします。
ウェーブが終了すると、アリーナの位置がリセットされることを忘れないでください！

  • Arena.width (読み取り専用) - ピクセル単位でサイズ変更後のアリーナの幅を返します。 プレイヤーと発射体(Projectile)の座標は、アリーナの中心が基準なので -Arena.width/2で左端、Arena.width/2で右端の座標を取得できます。

  • Arena.height (読み取り専用) - ピクセル単位でサイズ変更後のアリーナの高さを返します。 Arena.widthと同様に、-Arena.height/2で下端、Arena.height/2上端の座標を表します。

  • Arena.x (読み取り専用) - サイズ変更後の(?)アリーナの中心のx座標。

  • Arena.y (読み取り専用) - サイズ変更後の(?)アリーナの下端のy座標。 アリーナの中心のy座標はArena.y + Arena.height/2です。

    注: Arena.yはアリーナの外側の(白い部分)の下端を表します。 Arena.currentyよりも５ピクセル小さいです。[!?]
    (It will be 5 pixels (the arena's width) less than Arena.currenty.[原文])

  • Arena.currentwidth (読み取り専用) - 現在のアリーナの幅を取得します。 変更中の大きさを取得するという点でArena.widthと異なります。

  • Arena.currentheight (読み取り専用) - 現在のアリーナの高さを取得します。 変更中の大きさを取得するという点でArena.heightと異なります。
  • Arena.currentx (読み取り専用) - 現在のアリーナのx座標を取得します。 移動中の座標を取得するという点でArena.xと異なります。
  • Arena.currenty (読み取り専用) - 現在のアリーナのy座標を取得します。 移動中の座標を取得するという点でArena.yと異なります。

    NOTE: Arena.currenty is the position of the bottom of the inside (black part) of the Arena. It will be 5 pixels (the arena's width) greater than Arena.y.
    諦め(´・ω・`)

  • Arena.isResizing (読み取り専用) - アリーナの大きさが変更されている最中かどうか返します。
    : 0.2.1aには実装されていないので、次の評価式を使用してください。 -
    Arena.currentwidth ~= Arena.width or Arena.currentheight ~= Arena.height.

  • Arena.isMoving (読み取り専用) - アリーナが移動中かどうか返します。

  • Arena.isModifying (読み取り専用) - アリーナが移動中かサイズ変更中ならtrueを返し、それ以外ならfalseを返します。
  • Arena.Resize( width, height) - アリーナの大きさを変更します。Currently, monsters stay on top of the arena. This was going to be changed around the animation update.
    But, in CYF, you can use BindToArena to control that!

  • Arena.ResizeImmediate( width, height) - サイズ変更のアニメーション無しで、アリーナの大きさを変更します。

  • Arena.Move( x, y, movePlayer = true, immediate = false) - 現在のアリーナの位置に基づいて、 アリーナを移動させます。movePlayerをtrueに指定すると、プレイヤーがアリーナと共に動きます。 immediateをtrueに指定すると、アリーナがすぐに移動します。

  • Arena.MoveTo( x, y, movePlayer = true, immediate = false) - Moves the Arena based on the bottom-left corner of the window. Set movePlayer to true if you want the Player to move with the Arena and set immediate to true if you want to move the Arena immediately.

  • Arena.MoveAndResize( x, y, width, height, movePlayer = true, immediate = false) - Moves the Arena based on its current position and resizes it at the same time. Set movePlayer to true if you want the Player to move with the Arena and set immediate to true if you want to move the Arena immediately.

  • Arena.MoveToAndResize( x, y, width, height, movePlayer = true, immediate = false) - Move the Arena based on the bottom-left corner of the window and resizes it at the same time. Set movePlayer to true if you want the Player to move with the Arena and set immediate to true if you want to move the Arena immediately.

  • Arena.Hide() - アリーナを非表示にします。アリーナの境界の判定は残ります。

  • Arena.Show() - アリーナを表示させます。
    これはウェーブ終了時に自動的に呼び出されます。
