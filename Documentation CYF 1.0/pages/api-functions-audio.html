Audio オブジェクト [E/M/W] Audio オブジェクトを使用すると、ゲーム内の音楽(BGM)やサウンド(SE)の制御を行えます。使い方は次の通りです。

  • Audio.playtime - 現在流れている音楽の現在の再生時間を、秒単位で取得及び設定します。

  • Audio.totaltime - 現在流れている音楽の合計再生時間(長さ)を取得します。

  • Audio.Play() - 現在ロードされている音楽を再生します。戦闘開始時に自動的に呼び出されます。

  • Audio.Stop() - 音楽の再生を停止します。戦闘開始時にBGMを流したくないなら EncounterStarting()でこれを呼び出してください。

  • Audio.Pause() - 音楽の再生を一時停止します。

  • Audio.Unpause() - 一時停止した音楽の再生を再開します。

  • Audio.Volume( value) - 音楽の音量を設定します。valueは 0.0(ミュート)から 1.0(最大)の間でなければなりません。 0.75がデフォルトの音量です。

  • Audio.Pitch( value) - 音楽のピッチを設定します。1.0がデフォルトで 2.0で２倍の速度になります。負の値は音楽を逆再生にします。 valueの値は-3.0から3.0の間でなければなりません。

  • Audio.LoadFile( filename) - Audioフォルダにあるfilename.oggまたは filename.wavのファイルから音楽をロードし、すぐに再生します。 ロードのみを行う(すぐに再生したくない)場合は、直後にAudio.Stop()を呼び出してください。 引数にファイル拡張子は含めないでください。

  • Audio.PlaySound( filename, volume = 0.65) - Soundsフォルダにある filename.oggまたはfilename.wavのファイルからサウンドを再生します。 引数にファイル拡張子は含めないでください。

  • Audio.isPlaying - 音楽が再生されていればtrue、そうでなければfalseを返します。

  • Audio.StopAll() - 全ての音を止めます。

  • Audio.PauseAll() - 全ての音を一時停止します。

  • Audio.UnpauseAll() - 全ての一時停止している音を再開します。

  • Audio.SetSoundDictionary( key, value) - Adds a sound to the sound dictionary. Doing so allows you to change the name of the sounds played by the engine - like, for example, hurtsound or menumove.

    Setting key to "RESETDICTIONARY" will reset the entire sound dictionary to its original state.

  • Audio.GetSoundDictionary( key) - Returns the index of the sound in the sound dictionary. Returns the key itself if the key isn't in the dictionary.

  • Audio[ key], Audio[ key] = value - Fast way to use Audio.SetSoundDictionary / GetSoundDictionary.

これはオブジェクトであるため、[func]で直接呼び出すことはできません。 ダイアログの途中で音楽を止めたときなどは、独自の関数を定義する必要があります。

currentdialogue = {"but then I realized...\n[w:30][func:drama]the butler did it!!!"}

function drama()
    Audio.Stop()
    Audio.PlaySound("dramatic_sound_effect")
end
