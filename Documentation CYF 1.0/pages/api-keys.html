キーリスト



Input.GetKeyはUnityのキーコード(KeyCode)を使用します。


キーリストの完全なリストは こちら。



通常のアルファベッド:

(全てのアルファベッドは、キー名とCYFのキーIDと全く同じです。
例えば、"A"キーは、 CYFのキーIDではAになります。
Input.GetKey("A")を取得したとすると、 小文字の"a"が検出されることに気を付けてください。)


ファンクションキー:

(全てのファンクションキーも、キー名とCYFのキーIDと全く同じです
例えば、F12キーは、 CYFのキーIDではF12になります。
検出できるファンクションキーはF1から F15までです。)
