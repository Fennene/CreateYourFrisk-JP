Create Your Frisk のドキュメントへ ようこそ！



注意: このドキュメントには Undertaleの ネタバレが いたるところにあります。 Undertaleを プレイして クリアしていないのなら、 まず Undertaleを プレイすることを 推奨します！ 「ネタバレふんだんだけど！」 なんて ならないようにね。

多分 「早く作りたい！」 って思っているかも しれないから 前置きは短めに済ますね。 正直なところ、 CYFで作られたModを見て いじくりまわして どうなるのか 確かめるのが 手っ取り早いよ。 このドキュメントは CYFでできる全ての機能を 知りたい人用にあるよ。 何か手助けが必要なら、 /r/Unitale subreddit で 聞くといいかもよ。 (ただし 英語だけどね...)

Unitale v0.2.1aとの 互換性: 設定にある "retrocompatibility mode"を "Enable"(有効)にすると Unitale v0.2.1aで 作成されたModを 起動することができます！

メモ: Create Your Frisk v0.6.5 は Unity Personal 2018.2.13f1 で 作成されています。
GitHubのページは こちら。
(日本語翻訳バージョンは こちら。)


この横の ！マークは v0.6.5で 追加されたか 変更された コンテンツを 表します。

最後に このドキュメントの読み方を 読んでおく必要があります。

基本


操作方法
UnitaleとCYFでの 操作方法が 書かれたページ。

基本構成
Modフォルダーの構成と フォルダーやファイルの構成などが 書かれたページ。 最初に読むことを 推奨します。

Unity Setup (Optional)
Explains the process of setting up Unity for use with Create Your Frisk, how to use the editor to play mods, where files and folders are located, and how to export CYF builds.

予約変数
CYFが 独自に設定している 変数と その使い方が 書かれたページ。

用語
このドキュメントにおいて どういう言葉で説明されているのか 用語の一覧です。 例えば、 戦闘中にでてくる 白いボックス(文字が表示されたり 弾丸が飛び交うところ)は ここで "arena"(アリーナ) と呼ばる みたいな。 ドキュメントを読み進めていくうえで 分からなくならないように 読むためのページ。

API


テキスト コマンド
ダイアログボックス内の文字(または フレーバーテキスト)で 使用できる 特殊なコマンドが 書かれたページ。 エフェクトや 色付け、 声を変えたり スキップさせたり、 Luaコード内の 任意の関数を実行する コマンドなどがあります。

ゲーム イベント
このエンジンがLuaスクリプトから呼び出す関数の一覧。 どの時点で呼び出されるかが書かれてあります。 ACTコマンドやアイテム使用の直前直後や、戦闘が始まる前など、特定のタイミングに特定のイベントを発生させたい時に便利です。 必読すべし

関数 & オブジェクト
CYFのLuaスクリプトで使用できるほとんどの関数やオブジェクトの一覧とその使用方法など。 音楽の制御や、ゲームイベントやキーボード入力の確認の処理を行いたいですか？ スクリプト間で通信できるグローバルな変数？ プレイヤーがどれだけHPを持っているか確認したり、ダメージや回復する方法を知りたいですか？ それらを知りたいなら読むべきです! 発射体(Projectile)やスプライトの操作に関しては以下の別セクションに書かれています。

発射体(Projectile) 制御
このセクションでは、発射体(Projectile)の作成方法とその制御方法について書かれています。

Pixel-Perfect Collision システム
このセクションでは、CYFの"Pixel-Perfect Collision System"がどのようなものか、どのように動作するのか書かれています。

スプライト & アニメーション
このセクションでは、スプライトの作成方法とその制御方法について書かれています。 スプライトの機能を使うことによってアニメーションを作成できます。

テキスト オブジェクト
このセクションでは、テキストオブジェクトの作成方法とその制御方法について書かれています。 テキストオブジェクトを使用すると、任意の場所にフォントやバブルの有無などを設定したテキストを配置できます。

Shaders


Introduction
Here is where you can learn all about the shaders system added in Create Your Frisk v0.6.5! It requires a bit of setting up, but if you can handle it, shaders can be a fun visual addition to a mod.

The Shader Object
This is the actual object used from your battle's Lua scripts to apply and control shaders on a sprite or the entire screen.

Coding a Shader
Covers the process of creating and coding your own shaders for use with Create Your Frisk.

Overworld


Basics
This section covers how to set up CYF on your computer with Unity, and describes in general how the Overworld is set up. Read this first if you are making an overworld.

How to create a map
This is where you can learn how to make a map in CYF's Overworld, with Tiled2Unity in mind.

How to create an event
Events are all the different objects, interactables, and cutscenes that you can add to your map. This page explains how to set them up in the Unity editor.

How to animate an event
This smaller tutorial explains the CYFAnimator system in the Overworld, which removes the need to use coding to set up animations on your events.

How to create a shop
This page fully documents everything accessible within Shops you can set up in the Overworld.

Overworld Objects
These are all the different objects you can control from within your Event scripts, and how to use them. Want to show dialogue, save the game, move characters around, control the player's stats, mess with the screen, and so on? These are the pages to see!

A few things like the Audio object are carried over here from encounters. To see a list of all of them, see Overworld - Basics.
